# CSS杂记
## normalize.css
### 1. Normalize.css做了以下几件事：
-  **保护有用的浏览器默认样式**而不是完全去掉它们
-  **一般化的样式**：为大部分HTML元素提供
-  **修复浏览器自身的bug**并保证各浏览器的一致性
-  **优化CSS可用性**：用一些小技巧
-  **解释代码**：用注释和详细的文档来
### 2. 使用方式
[git链接](https://github.com/necolas/normalize.css)
#### npm
```shell
npm install --save normalize.css
```
